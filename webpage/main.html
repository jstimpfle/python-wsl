python-wsl

python-wsl is a python3 library for interfacing with WSL databases.

Download

The easiest way to get python-wsl is to install it with pip3: If you have it, just say pip3 install --user wsl. If you need the newest development version or want to make changes, you can also clone the git repo on github.

Documentation

Online documentation. The documentation can also be built from the project sources.
